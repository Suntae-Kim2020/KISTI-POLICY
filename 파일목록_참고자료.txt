[소스코드]

  compute.py                    데이터 전처리 스크립트.
                                KISTEP 폴더의 pickle·JSON을 로딩하여
                                섹션별 통계를 계산하고 data_cache.json 생성.
                                실행 시 ~3-4GB RAM 필요, 약 2분 소요.

  app.py                        Flask 웹서버 (port 5002).
                                data_cache.json을 로딩하여 API로 서빙.

  scan_kbsi_induced.py          KBSI 유발논문 탐색 스크립트.
                                WoS 원시 데이터에서 KBSI 인프라 키워드를
                                검출하여 kbsi_induced_papers.json 생성.

  templates/dashboard.html      싱글 페이지 대시보드 (Chart.js 4.4.7, 다크 테마).
                                32개 페이지, ~80개 차트, 인사이트 설명 포함.

  exclusions.json               대시보드에서 제외한 논문 UT 목록.
                                kisti / induced / kbsi / kbsi_induced 4개 키.

2. KISTEP 폴더 의존 파일 (경로: /Users/kimsuntae/KISTEP/)
────────────────────────────────────────────────────────────────────────────────

[pickle 데이터 — compute.py가 로딩하는 소스 데이터]

  wos_data.pkl                  WoS 논문 메타데이터 (~931MB, 약 96만건).
                  (931MB)       컬럼: UT, PY, DT, TC, SO, std_field 등.
                                preprocess_wos.py로 WoS TXT에서 생성.

  wos_institutions.pkl          WoS 기관 데이터 (~580MB, 약 281만건).
                  (580MB)       컬럼: UT, org_alias, institution_type_7,
                                country_code 등.
                                논문-기관 매핑에 사용.

  jcr_jif.pkl                   JCR Journal Impact Factor 데이터 (~35MB).
                  (35MB)        저널별 JIF, JIF Quartile 정보.
                                학술지 분석(sec1_5, sec4_5) 및
                                저널 분위(Q1-Q4) 분석(sec6_4)에 사용.

  esi_journal_map.pkl           ESI 저널-분야 매핑 (~312KB).
                  (312KB)       ISSN 기반으로 ESI 22개 분야를 할당.
                                std_field 재매핑에 사용.

  kr_journals.pkl               한국 저널 목록 (~15KB).
                  (15KB)        국내 학술지 식별에 사용.

  multi_reclass.pkl             Multidisciplinary 재분류 데이터 (~5B).
                  (5B)          현재 CR 데이터 부재로 미적용 (빈 파일).

[JSON 데이터 — 유발논문 목록]

  kisti_induced_papers.json     KISTI 유발논문 UT 목록 (~9.5MB, 6,307건).
                  (9.5MB)       WoS FU/FX 필드에서 KISTI 인프라 키워드
                                (KSC, NURION, KREONET, EDISON 등) 검출.
                                인프라 유형별 분류 포함.

  kbsi_induced_papers.json      KBSI 유발논문 UT 목록 (~7.3MB).
                  (7.3MB)       scan_kbsi_induced.py로 생성.
                                WoS FU/FX에서 KBSI 인프라 키워드 검출.

  kisti_induced_summary.json    KISTI 유발논문 요약 통계 (~545B).
                  (545B)        인프라별 건수 요약.

[스크립트]

  preprocess_wos.py             WoS 원시 TXT 파일을 파싱하여
                  (67KB)        wos_data.pkl, wos_institutions.pkl 생성.
                                KISTI_Policy 프로젝트의 원본 데이터 생성기.

  build_report.py               KISTEP 보고서 생성 스크립트 (~1.3MB).
                  (1.3MB)       KISTI_Policy와 별도 프로젝트이나
                                동일 pickle 데이터를 공유.

  app.py                        KISTEP 대시보드 Flask 서버 (~27KB).
                  (27KB)        KISTI_Policy와 별도 운영 (다른 포트).

  config.json                   KISTEP 프로젝트 설정 파일 (~133B).
                  (133B)
